What's in Your Weekly Report

This is an actual sample report from a real diagnostic run. Below it, we explain what each section means in plain language so you can see exactly what your facility manager or building engineer would receive every week.

  • ✅Passing systems are listed first — so you can quickly confirm what's working correctly before focusing on issues.
  • ⚠️Issues are prioritized by severity — critical problems appear at the top, minor observations at the bottom.
  • 📅Days open is tracked — so you can see if an issue is new this week or has been present (and possibly ignored) for weeks.
  • 📋Notes explain the problem — in plain language that doesn't require an HVAC certification to understand.
  • 🎯Equipment IDs tie to your BAS — so your controls contractor can find the exact device immediately.
